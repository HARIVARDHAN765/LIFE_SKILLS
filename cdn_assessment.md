# Content Delivery Network (CDN)

## Overview

A Content Delivery Network (CDN) has many edge servers located in different places. Instead of sending data from a single server, these edge servers help deliver data quickly to users. These servers store cached copies of content, and when a user makes a request, the nearest server sends the data. This helps reduce latency and improve speed.

## Benefits

* Reduces latency by sending data from nearest servers  
* Improves performance and loads faster 
* It Provides load balancing for handling large traffic  
* It gives security against DDoS attacks
